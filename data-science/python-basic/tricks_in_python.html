Thủ thuật trong Python#

Nên định nghĩa kiểu giá trị của biến

n: int = 8
s: str = "Chuỗi"

Sử dụng ... thay cho pass để trống một khối lệnh hoặc raise lỗi NotImplementedError

def my_function():
    ...

Nên viết tài liệu cho hàm xử lý, giải thích các biến

Sử dụng * như một tham số của hàm để bắt người dùng nhập vào phải chứa thông tin tham số hàm

def my_func(s, *, v):
    ...
    
my_func(s = '', v = '')

custom_folder: str = "New folder"

with open (path, 'r') as f:

Walrus Operator#

Toán tử gán trong python ‘:=’

numbers = [2, 8, 0, 1, 1, 9, 7, 7]

description = {
    "length": (num_length := len(numbers)),
    "sum": (num_sum := sum(numbers)),
    "mean": (num_mean := num_sum / num_length),
}

print(num_length)
print(num_sum)
print(num_mean)
print(description)

8
35
4.375
{'length': 8, 'sum': 35, 'mean': 4.375}

String methods#

Có thể sử dụng ‘_’ để phân tách hàng của số để dễ đọc

n: int = 1_000_000_000
print(f"{n}")

1000000000

Sử dụng f-string để biểu diễn số đẹp hơn để phân tách các hàng của số

n: int = 1_000_000_000
print(f"{n:_}")
print(f"{n:,}")

1_000_000_000
1,000,000,000

Sử dụng ‘<’, ‘>’ hay ‘^’ sau ‘:’ của giá trị để biểu diễn chuỗi

s: str = 'I Love U'
print(f"{s:|<20}")
print(f"{s:_>20}")
print(f"{s:#^20}")

I Love U||||||||||||
____________I Love U
######I Love U######

Sử dụng f-string để biểu diễn các số lượng chữ số sau dấy ‘.’ của float (Kết hợp với ‘,’ để phân tách các hàng của số)

f: float = 123123.456
print(f"{f:,.2f}")

123,123.46

Sử dụng f-string để in kết quả của phép toán bằng cách thêm dấu ‘=’ vào phép toán

a: int = 5
b: int = 3

print(f"{a + b = }")

a + b = 8

Sử dụng center để căn giữa string

text = "I Love U"

print(text.center(30, '.'))

...........I Love U...........

Sử dụng ljust rjust để thêm ký tự bên phải, trái của chuỗi

text = "I Love U"

print(text.ljust(20, '.'))
print(text.rjust(20, '.'))

I Love U............
............I Love U

Set operations#

Các phép toán trên set

set_a: set[int] = {1, 2, 3, 4, 5}
set_b: set[int] = {4, 5, 6, 7, 8}

Phép OR

print(set_a | set_b)

{1, 2, 3, 4, 5, 6, 7, 8}

Phép trừ

print(set_a - set_b)
print(set_b - set_a)

{1, 2, 3}
{8, 6, 7}

Phép AND

print(set_a & set_b)

{4, 5}

Phép XOR

print(set_a ^ set_b)

{1, 2, 3, 6, 7, 8}
