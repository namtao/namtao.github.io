Module trong Python#

Định nghĩa Module trong Python#

Module trong Python được sử dụng để phân loại code thành các phần nhỏ hơn liên quan với nhau. Hay nói cách khác, Module giúp bạn tổ chức Python code một cách logic để giúp bạn dễ dàng hiểu và sử dụng code đó hơn.

Module là một đối tượng với các thuộc tính mà bạn có thể đặt tên tùy ý và bạn có thể gắn kết và tham chiếu.

Về cơ bản, một Module là một file, trong đó các lớp, hàm và biến được định nghĩa. Tất nhiên, một Module cũng có thể bao gồm code có thể chạy.

Bạn theo dõi qua ví dụ sau: Nếu nội dung của một quyển sách không được lập chỉ mục hoặc phân loại thành các chương riêng, thì quyển sách này có thể trở nên nhàm chán và gây khó khăn cho độc giả khi đọc và hiểu nó. Tương tự, Module trong Python là các file mà có các code tương tự nhau, hay có liên quan với nhau. Chúng có lợi thế sau:

  • Khả năng tái sử dụng: Module có thể được sử dụng ở trong phần Python code khác, do đó làm tăng tính tái sử dụng code.

  • Khả năng phân loại: Các kiểu thuộc tính tương tự nhau có thể được đặt trong một Module.

Câu lệnh import#

Để import một Module, bạn có thể sử dụng một trong ba cách dưới đây:

Sử dụng lệnh import trong Python#

Bạn có thể sử dụng bất cứ source file nào dưới dạng như một Module bằng việc thực thi một lệnh import trong source file khác. Cú pháp của lệnh import là:

import Module1[, Module2[,... ModuleN]

Giả sử mình có đoạn code trong một file addition.py:

def add(a,b):
    c = a+b
    print(c)
    return

Lệnh import được sử dụng như sau với file này:

import data_module_test.addition as addition

addition.add(10,20)
addition.add(30,40)

Để import nhiều Module, bạn sử dụng cách như trong ví dụ sau:

import data_module_test.greeting as greeting, data_module_test.class_information as class_information

greeting.hello("Dang")
class_information.show_information("Dang")

Sử dụng lệnh from…import trong Python#

Lệnh from…import được sử dụng để import thuộc tính cụ thể từ một Module. Trong trường hợp mà bạn không muốn import toàn bộ Module nào đó thì bạn có thể sử dụng lệnh này. Cú pháp của lệnh from…import là:

from modname import name1[, name2[, ... nameN]]

Dưới đây là ví dụ

from data_module_test.greeting import hello

hello("Dang")

Hello, Dang

Sử dụng lệnh from…import* trong Python#

Sử dụng lệnh này, bạn có thể import toàn bộ Module. Do đó bạn có thể truy cập các thuộc tính trong Module này. Cú pháp của lệnh là:

from modname import *

from data_module_test import *
addition.add(10,20)
greeting.hello("Dang")

30
Hello, Dang

Đổi tên module đã nhập#

Bạn có thể đổi tên Module bạn đang nhập, điều này hữu ích trong trường hợp bạn muốn đặt tên có ý nghĩa hơn cho mô-đun hoặc tên Module quá lớn để sử dụng nhiều lần. Bạn có thể sử dụng từ khóa as để đổi tên nó.

# Ví dụ
import data_module_test.class_information as class_inf
print(class_inf.show_information("Dang"))

None

Package module#

Có thể gom nhiều module .py vào một thư mục và tên thư mục là tên của package và tạo một file init.py trong thư mục này.

Như vậy, cấu trúc thư của một package sẽ như sau:

|--	mypack
|			|--	__init__.py
|			|--	mymodule1.py
|			|--	mymodule2.py
|

Có thể sử dụng addition theo cú pháp import sau:

import	mypack.mymodule1

hoặc

import	mypack.mymodule1	as	mymodule1

hoặc

import	mypack.mymodule1	as	mod

# Ví dụ
import data_module_test.greeting

Về cơ bản, một Package là một tập hợp các Module, sub-package, … tương tự nhau. Đó là một cấu trúc có thứ bậc của thư mục và file.

Dưới đây là các bước để tạo và import một Package:

  1. Tạo một thư mục.

  2. Đặt các Module khác nhau bên trong thư mục.

  3. Tạo một __init__.py file để xác định các thuộc tính trong mỗi Module.

  4. Cuối cùng bạn import package này và sử dụng các thuộc tính đó bởi sử dụng package.

Ví dụ xây dựng Module#

Ta sẽ tiến hành tạo một module xử lý đọc và ghi file đơn giản

Bước 1: Tạo Package#

import os
os.mkdir("XuLyFile")

Bước 2: Tạo module#

Lần lượt tạo các module xử lý file trong Package

# File luu_file.py
file_content = """\
def luu_file(du_lieu, ten_file, mode):
    with open(ten_file, mode) as file:
        file.write(str(du_lieu))
"""

file_path = "XuLyFile/luu_file.py"

with open(file_path, "w") as f:
    f.write(file_content)

# File doc_file.py
file_content = """\
def doc_file(file_path,mode):
    with open(file_path, mode) as f:
        data = f.read()
    return data

"""

file_path = "XuLyFile/doc_file.py"

with open(file_path, "w") as f:
    f.write(file_content)

luu_file(["1","MetaMind"],"test.txt","w")
doc_file("test.txt","r")

"['1', 'MetaMind']"

Lưu các file doc_file.py và luu_file.py vào folder XuLyFile

Bước 3: Tạo __init__.py#

File __init__ file giống như một file được sử dụng để xem xét các thư mục trên disk dưới dạng package của Python. Về cơ bản, file này được sử dụng để khởi tạo các Package trong Python.

Thông thường, nếu ta không có file __init__.py, ta sẽ phải import như thế này:

from XuLyFile.luu_file import luu_file
from XuLyFile.doc_file import doc_file

Đặt trường hợp nếu có 3 vạn 9 nghìn cái file mà lại đi import thủ công vậy thì rất mất thời gian. Vậy nên ta cần tổ chức lại toàn bộ cấu trúc thư viện thông qua file __init__.py như sau

file_content = """\
from .luu_file import luu_file
from .doc_file import doc_file
"""

file_path = "XuLyFile/__init__.py"

with open(file_path, "w") as f:
    f.write(file_content)

Lưu ý: Dấu . trong câu lệnh from .luu_file import luu_file hay from .doc_file import doc_file có nghĩa là import từ module hiện tại. Khi bạn sử dụng dấu . như vậy, Python hiểu rằng bạn muốn import từ module cùng mức độ với module hiện tại.

Bước 4: Import và dùng thôi#

import XuLyFile
# Sử dụng các hàm
luu_file(["3", "MetaMind 3"], "test.txt", "w")
print(doc_file("test.txt", "r"))

['3', 'MetaMind 3']
