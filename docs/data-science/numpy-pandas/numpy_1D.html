Chuẩn bị

# Import the libraries

import time
import sys
import numpy as np

import matplotlib.pyplot as plt
%matplotlib inline

# Plotting functions

def Plotvec1(u, z, v):

    ax = plt.axes()
    ax.arrow(0, 0, *u, head_width=0.05, color='r', head_length=0.1)
    plt.text(*(u + 0.1), 'u')

    ax.arrow(0, 0, *v, head_width=0.05, color='b', head_length=0.1)
    plt.text(*(v + 0.1), 'v')
    ax.arrow(0, 0, *z, head_width=0.05, head_length=0.1)
    plt.text(*(z + 0.1), 'z')
    plt.ylim(-2, 2)
    plt.xlim(-2, 2)

def Plotvec2(a,b):
    ax = plt.axes()
    ax.arrow(0, 0, *a, head_width=0.05, color ='r', head_length=0.1)
    plt.text(*(a + 0.1), 'a')
    ax.arrow(0, 0, *b, head_width=0.05, color ='b', head_length=0.1)
    plt.text(*(b + 0.1), 'b')
    plt.ylim(-2, 2)
    plt.xlim(-2, 2)

Numpy là gì?

Mảng numpy tương tự như list. Nó thường cố định về kích thước và mỗi phần tử là cùng kiểu. Chúng ta có thể truyền một list vào mảng numpy bằng cách nhập numpy đầu tiên:

# import numpy library

import numpy as np

Sau đó, chúng ta ép kiểu list như sau:

# Create a numpy array

a = np.array([0, 1, 2, 3, 4])
a

array([0, 1, 2, 3, 4])

Mỗi phần tử có cùng kiểu, trong trường hợp này là các số nguyên:

Như với list, chúng ta có thể truy cập từng phần tử thông qua dấu ngoặc vuông:

# Print each element

print("a[0]:", a[0])
print("a[1]:", a[1])
print("a[2]:", a[2])
print("a[3]:", a[3])
print("a[4]:", a[4])

a[0]: 0
a[1]: 1
a[2]: 2
a[3]: 3
a[4]: 4

Kiểu

Nếu kiểm tra kiểu của mảng, ta được numpy.ndarray:

# Check the type of the array

type(a)

numpy.ndarray

Vì các mảng không chứa dữ liệu cùng kiểu, chúng ta có thể sử dụng thuộc tính “dtype” để lấy Kiểu dữ liệu của các phần tử của mảng. Trong trường hợp này là số nguyên 64 bit:

# Check the type of the values stored in numpy array

a.dtype

dtype('int32')

Chúng ta có thể tạo một mảng numpy với các số thực:

# Create a numpy array

b = np.array([3.1, 11.02, 6.2, 213.2, 5.2])

Khi kiểm tra kiểu của mảng, ta được numpy.ndarray:

# Check the type of array

type(b)

numpy.ndarray

Nếu kiểm tra thuộc tính dtype, chúng ta sẽ thấy float64 vì các phần tử không phải là số nguyên:

# Check the value type

b.dtype

dtype('float64')

Gán giá trị

Chúng ta có thể thay đổi giá trị của mảng, hãy xem xét mảng c:

# Create numpy array

c = np.array([20, 1, 2, 3, 4])
c

array([20,  1,  2,  3,  4])

Chúng ta có thể thay đổi phần tử đầu tiên của mảng thành 100 như sau:

# Assign the first element to 100

c[0] = 100
c

array([100,   1,   2,   3,   4])

Chúng ta có thể thay đổi phần tử thứ 5 của mảng thành 0 như sau:

# Assign the 5th element to 0

c[4] = 0
c

array([100,   1,   2,   3,   0])

Slicing

Giống list, chúng ta có thể cắt mảng numpy, chọn các phần tử từ 1 đến 3 và gán nó vào mảng numpy d mới như sau:

# Slicing the numpy array

d = c[1:4]
d

array([1, 2, 3])

Chúng ta có thể gán các chỉ mục tương ứng cho các giá trị mới như sau:

# Set the fourth element and fifth element to 300 and 400

c[3:5] = 300, 400
c

array([100,   1,   2, 300, 400])

Gán giá trị với List

Tương tự, chúng ta có thể sử dụng một list để chọn một chỉ mục cụ thể. List ‘select’ chứa một số giá trị:

# Create the index list

select = [0, 2, 3]

Chúng ta có thể sử dụng list như một đối số trong ngoặc. Đầu ra là các phần tử tương ứng với chỉ số cụ thể:

# Use List to select elements

d = c[select]
d

array([100,   2, 300])

Chúng ta có thể gán các phần tử được chỉ định cho một giá trị mới. Ví dụ, chúng ta có thể gán các giá trị cho 100 000 như sau:

# Assign the specified elements to new value

c[select] = 100000
c

array([100000,      1, 100000, 100000,    400])

Các thuộc tính khác

Hãy xem lại một số thuộc tính cơ bản của mảng bằng cách sử dụng mảng a:

# Create a numpy array

a = np.array([0, 1, 2, 3, 4])
a

array([0, 1, 2, 3, 4])

Thuộc tính size là số phần tử trong mảng:

# Get the size of numpy array

a.size

5

Hai thuộc tính tiếp theo sẽ có ý nghĩa hơn khi chúng ta được kích thước cao hơn nhưng hãy xem lại chúng. Thuộc tính ndim đại diện cho số lượng chiều hoặc thứ hạng của mảng, trong trường hợp này là:

# Get the number of dimensions of numpy array

a.ndim

1

Thuộc tính shape là một tập hợp các số nguyên cho biết kích thước của mảng theo từng chiều:

# Get the shape/size of numpy array

a.shape

(5,)

# Create a numpy array

a = np.array([1, -1, 1, -1])

# Get the mean of numpy array

mean = a.mean()
mean

0.0

# Get the standard deviation of numpy array

standard_deviation=a.std()
standard_deviation

1.0

b = np.array([-1, 2, 3, 4, 5])
b

array([-1,  2,  3,  4,  5])

# Get the biggest value in the numpy array

max_b = b.max()
max_b

5

# Get the smallest value in the numpy array

min_b = b.min()
min_b

-1

Hoạt động mảng Numpy

Cộng mảng

Xem xét mảng numpy u:

u = np.array([1, 0])
u

array([1, 0])

Xem xét mảng numpy v:

v = np.array([0, 1])
v

array([0, 1])

Chúng ta có thể cộng hai mảng và gán nó cho z:

# Numpy Array Addition

z = u + v
z

array([1, 1])

Phép toán tương đương với phép cộng vectơ:

# Plot numpy arrays

Plotvec1(u, z, v)

Nhân mảng

Hãy xem xét mảng numpy vectơ y:

# Create a numpy array

y = np.array([1, 2])
y

array([1, 2])

Chúng ta có thể nhân mọi phần tử trong mảng với 2:

# Numpy Array Multiplication

z = 2 * y
z

array([2, 4])

Điều này tương đương với việc nhân một vectơ với một đại lượng vô hướng:

Tích của 2 mảng numpy

Hãy xem xét mảng u sau:

# Create a numpy array

u = np.array([1, 2])
u

array([1, 2])

Hãy xem xét mảng sau v:

# Create a numpy array

v = np.array([3, 2])
v

array([3, 2])

Tích của hai mảng numpy u và v được cho bởi:

# Calculate the production of two numpy arrays

z = u * v
z

array([3, 4])

Tích vô hướng

Tích vô hướng của hai mảng numpy u và v được cho bởi:

# Calculate the dot product

np.dot(u, v)

7

Cộng hằng số 1 vào mỗi phần tử trong mảng:

# Add the constant to array

u + 1

Hàm số

Chúng ta có thể truy cập giá trị của pi trong numpy như sau:

# The value of pi

np.pi

3.141592653589793

Chúng ta có thể tạo mảng numpy sau trong Radian:

# Create the numpy array in radians

x = np.array([0, np.pi/2 , np.pi])

Chúng ta có thể vận dụng hàm sin cho mảng x và gán các giá trị cho mảng y; điều này áp dụng hàm sin cho mỗi phần tử trong mảng:

# Calculate the sin of each elements

y = np.sin(x)
y

array([0.0000000e+00, 1.0000000e+00, 1.2246468e-16])

Linspace

Linspace là một hàm hữu ích để vẽ các hàm toán học. Linspace trả về các số cách đều nhau trong một khoảng được chỉ định. Chúng ta chỉ định điểm bắt đầu và kết thúc của dãy. Tham số “num” cho biết số lượng mẫu cần tạo, trong trường hợp này là 5:

# Makeup a numpy array within [-2, 2] and 5 elements

np.linspace(-2, 2, num=5)

array([-2., -1.,  0.,  1.,  2.])

Nếu thay đổi tham số num thành 9, chúng ta được 9 số cách đều nhau trong khoảng từ -2 đến 2:

# Makeup a numpy array within [-2, 2] and 9 elements

np.linspace(-2, 2, num=9)

array([-2. , -1.5, -1. , -0.5,  0. ,  0.5,  1. ,  1.5,  2. ])

Chúng ta có thể sử dụng hàm linspace để tạo 100 mẫu cách đều nhau từ khoảng 0 đến 2π:

# Makeup a numpy array within [0, 2π] and 100 elements

x = np.linspace(0, 2*np.pi, num=100)

Chúng ta có thể vận dụng hàm sin cho từng phần tử trong mảng x và gán nó cho mảng y:

# Calculate the sine of x list

y = np.sin(x)

# Plot the result

plt.plot(x, y)

Quiz về Mảng Numpy 1D

Thực hiện phép trừ vectơ sau trong numpy: u-v

u = np.array([1, 0])
v = np.array([0, 1])
print(u - v)

[ 1 -1]

Nhân mảng numpy z với -2:

z = np.array([2, 4])

print(z * (-2))

[-4 -8]

Xem xét list [1, 2, 3, 4, 5] và [1, 0, 1, 0, 1], sau đó truyền cả hai danh sách sang một mảng numpy rồi nhân chúng với nhau:

lst1 = [1, 2, 3, 4, 5]
lst2 = [1, 0, 1, 0, 1]

np.array(lst1) * np.array(lst2)

array([1, 0, 3, 0, 5])

Chuyển đổi list [-1, 1] và [1, 1] thành mảng numpy a và b. Sau đó, vẽ các mảng dưới dạng vectơ bằng cách sử dụng hàm Plotvec2 và tìm tích vô hướng:

lst1 = [-1, 1]
lst2 = [1, 1]
a = np.array(lst1)
b = np.array(lst2)
Plotvec2(a, b)

print(np.dot(a, b))

0
